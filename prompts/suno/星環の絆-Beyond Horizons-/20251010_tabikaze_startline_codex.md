# Suno Prompt – CNP TCG Adventure Anthem "旅風のスタートライン"

**Title**
旅風のスタートライン (Startline of Roaming Winds)

---

## 楽曲基本情報
- **楽曲種類**: オープニングテーマ
- **対象長さ**: TV size (1:30-1:40)
- **BPM / Key**: 148 BPM / D major

## テーマ・世界観設定
- **主要舞台**: 四つの大地を巡る長旅（海・炎・森・都市・軌道コロニー）
- **中心キャラクター**: 旅立つ仲間たち（グループ全員を想定）
- **物語の局面**: 未知の世界へ飛び込み、それぞれの地で解放と再会を描く冒険の開幕
- **感情・雰囲気**: 疾走感、希望、敗北を燃料に変える決意、仲間との連帯

## 音楽ジャンル・スタイル
- **基本ジャンル**: ハイテンポJ-Rock
- **サブジャンル**: アニメオープニング系アンセム、エモーショナル・ポストロック要素
- **主要楽器**: ツインエレキギター、ベース、ドラム、シンセパッド、シーケンスノイズ、クラップ／ハンドクラップ
- **音響要素**: ギターフィードバック、ノイズリバース、ギャングシャウト、アナログディレイ

## ボーカル設定
- **メイン**: 女性ハイトーン（ギラついたアタック＋伸びやかなファルセット）
- **ハーモニー**: サビで男女混成ユニゾン＋オクターブダブル、ブリッジでギャングシャウト
- **言語**: 日本語主体＋英語タグ（短いフレーズのみ）
- **ニュアンス**: イントロ囁き→ヴァース勢い→サビで空へ跳躍→ブリッジ半テン→ラストサビ倍速感

## 世界観根拠（カード・資料）
- `BT1-90 / ヤーマ / プロローグ⑧: ホールを抜け未知の世界へ飛び出し冒険が始まる`
- `BT1-28 / 戦火のアクアノーヴァ / 青い都市が圧政下でも解放を望む描写`
- `BT2-27 / 炎の嵐 / 火山の大陸が侵略と火の嵐に包まれる状況`
- `BT1-109 / 到着！メテオラス！ / 森の大地で三勢力の混迷が続いている`
- `BT1-82 / 到着！ミッドガン！ / 研究都市で機兵が侵略に抗い革新が続く`
- `BT2-61 / 逃げて来た貿易商人 / 貿易コロニーの閉鎖と住民の困窮`
- `CNP_TCG_STORY_ANALYSIS.md / 物語テーマ「敗北が糧となる世界」`

## メッセージ・キーワード
- **中心メッセージ**: 未知の地での敗北や痛みを糧にし、仲間と共に旅を続けることで新しい光を掴む
- **重要キーワード**: 旅風 / スタートライン / コンパス / 翼 / Horizon / spark / defeat as fuel / hand in hand
- **避けるべき表現**: キャラクター固有名詞・大陸固有名詞の直接使用
- **世界観用語の扱い**: 色や象徴（海の蒼、火の紅、森の翠、都市の黄金、軌道の紫）で暗示

## 楽曲構成
- **基本構成**: Intro → Verse → Pre-Chorus → Chorus → Verse2 → Pre-Chorus → Chorus → Bridge → Final Chorus → Outro
- **転調**: なし（終盤でサブドミナントリフト＋ハイトニック装飾）
- **テンポ変化**: ブリッジで半テン、ラストサビで倍速シンバル

## 制作条件・制約
- **Suno向け最適化**: 4セクション圧縮、全体1000文字以内、明確なスタイルタグ
- **言語バランス**: 日本語メイン＋英語フック（3～4語）
- **繰り返し要素**: コーラス冒頭の "We fly, we run" をフック化
- **技術的制約**: ハイテンポでもメロ維持のためメロディレンジをE4-A5付近に設定

## 既存楽曲との関係性
- **シリーズ位置づけ**: 第1期OP「未知なる旅路へ」との連続性を保ちつつ、さらに疾走感を高めた第2期冒険アンセム
- **音楽的関係**: 既存OPよりもタイトなビートとギターリフでスピード感を強調
- **ストーリー的関係**: 同じ旅路の次ステージで、各地の戦いへ跳び出す直前の昂揚を描写
