# Suno Prompt – CNP TCG ED2「夜空の約束」

**Title**
Yozora no Yakusoku (Promise Under the Night Sky)

---

## 📋 楽曲基本情報

- **楽曲タイトル（日本語）**: 夜空の約束
- **楽曲タイトル（英語）**: Promise Under the Night Sky
- **楽曲種類**: アニメエンディングテーマ（第2クール / 続編）
- **対象長さ**: TV size (1:30-1:45)
- **位置付け**: 既存ED「帰り道の空」の続編。戦いの後の静かな夜、星空を見上げながら仲間を想い明日への希望を抱く

---

## 🎵 テーマ・世界観設定

### **ストーリー背景**
第2クールでは、CNPメンバーたちが各大陸に散らばり、それぞれの場所で試練を乗り越えて成長する物語。一日の戦いを終えた夜、同じ星空を見上げながら離れた仲間を想い、明日への小さな希望を胸に眠りにつく。

### **世界観根拠（CNP TCG公式設定より）**

#### **各大陸での活動（公式設定）**
- **リーリー・ルナ**: アクアノーヴァ（青い海）で戦い、ミカヅキから神器奪還
- **ナルカミ・マカミ**: カグツチ（炎の大陸）で竜人とオーガの対立調停
- **オロチ・ヤーマ**: メテオラス（緑の森）で種族争いの解決を模索
- **トワ・セツナ**: ミッドガン（黄金の都市）で学派と反学派の争いに巻き込まれる
- **ミタマ・リサ**: 貿易コロニー（宇宙）で脱出を目指す

#### **絆のテーマ（根拠カード）**
- **BT2-11 / 海の支配者 ミナモ**: 「仲間との絆が力になる」
- **BT1-33 / ナルカミ**: 「真面目で責任感が強いしっかり者。仲間を守る」
- **BT1-46 / ルナ**: 「静かに行動し、仲間を見守る」
- **公式PV第2弾**: 「突如閉ざされた貿易コロニー → 彼らは合流するためミッドガンへの降下を試みる」

### **音楽コンセプト**
- **第1ED「帰り道の空」との対比**:
  - 第1ED: 故郷への帰還願望・旅の終わり・温かな郷愁
  - 第2ED: 今いる場所での安らぎ・夜の静けさ・明日への希望
- **アコースティック継承**: 既存EDと同じく温かみのある音色
- **ギター弾き語り重視**: よりシンプルで親密な響き、夜の焚き火を囲むような雰囲気

## 🎸 音楽設定

### **基本情報**
- **ジャンル**: アコースティック・フォークバラード
- **スタイル**: ギター弾き語り × 優しい女性ボーカル
- **BPM**: 75-85（ゆったりとしたテンポ）

- **メイン**: フィンガーピッキング・アコースティックギター

- **後半のみ**: ストリングス（最終サビで温かく包み込む）

## 🎼 Sunoプロンプト（最適化版・870文字以内）

**Style/Refs**
Acoustic folk ballad intimate singer-songwriter. Fingerpicked guitar w/minimal accompaniment. Warm night atmosphere anime ED. 1:30-1:45 TV. 75-85 BPM, D major. Structure: Intro→V→C→V2→Pre→C→Bridge→Final C (key up)→Outro.

**Vocal tone**
Warm gentle female, intimate breathy delivery, soft storytelling. Whisper verses, stronger chorus w/light harmonies. Natural vibrato, conversational. Tender reassuring, minimal effects.

**Arrangement**
Intro fingerpicked guitar only night mood, V1 guitar+soft vocal intimate, C warmer guitar+harmonies, V2 add piano right, Pre building gently, C2 fuller guitar+brush drums+piano, Bridge quiet guitar+emerging strings, Final C key up+string quartet+layered vocals, Outro fade guitar+humming.

**Production**
Close-mic vocal, room reverb. Centered guitar slight stereo fills, piano 30% right, strings wide. Analog feel, minimal compression. −16 to −18 LUFS. Fade ≤1:45. Campfire starry night, voice+guitar focus.

## 📚 制作背景・根拠

### **第1EDとの関係性**
