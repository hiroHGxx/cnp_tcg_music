# 3rd Works – 五彩リゾナンス企画ノート

## アルバム概要
- **アルバム名（仮）**: Prismatica☆Destiny
- **ユニット名**: 五彩リゾナンス (Gosai Resonance)
- **コンセプト**: カグツチ・ミッドガン・メテオラス・貿易コロニー・アクアノーヴァの協力者5人が、敗北を糧に未来を繋ぐ次元横断音楽ユニットとして結成。各大陸の民に希望のライブ配信を届け、セイドウに奪われた日常を音楽で取り戻す。
- **サウンド指針**: 2020年代の現代的エレクトロポップ・K-Pop・Perfume系テクノポップを軸に、デジタルサウンドで統一した音楽アルバム。

## メンバープロフィール整理
- **ツミカ (BT1-58 / BT2-25)**: 内気だが聡明な竜人少女。炎神を守るため争いの扇動も辞さない情熱。→ 炎×ダンスロック要素、煽り担当。
- **ミライ (BT2-34)**: ミッドガン反学派の希望。母を止めるという静かな決意とテクノロジーへの精通。→ テクノポップ、ポジティブな未来志向のリーダーボーカル。
- **クルミ (BT1-115)**: 天然で明るいフェアリー。ボディランゲージで種族を超えて導く。→ グリーン×フォレストポップ、癒やし系メイン。
- **リサ (BT2-81)**: 謎多きコロニーの少女。潜伏とハッキングで脱出策を探す。→ 宇宙×シンセウェーブ、低音ラップ＆クールなセクション。
- **ミナモ (BT1-29 / BT2-11)**: 海の生き物と意思疎通するメタモル。怒涛の陽動作戦を率いた。→ アクアトランス、コーラスリーダー。

## 物語フレーム
1. セイドウの支配下で各地が暗いムードに → 5人が連絡網を構築。
2. 「ライブ配信＝レジスタンスの暗号」。音楽を通じて抵抗勢力を鼓舞。
3. 失敗や敗北もMCに織り込んで共有、共感の輪を広げる。
4. 最終曲で、次元ゲート越しにCNP本隊とシンクロする布石を残す。

## 制作上の重要ルール
- Sunoプロンプト: Stylesセクションは**500-600文字目標**（1000文字制限対策）
- ボーカル: 特に指定がない場合、メインボーカルは女性
- メタタグ必須: 歌詞の各セクションに `[セクション名 – 楽器/雰囲気, 声質]` 形式
- キャラクター名: 歌詞内に記載すると歌ってしまうため削除
- サウンド方針: 2020s デジタルサウンド、80-90s アナログ感は排除
- CNP世界観: 固有名詞は控えめに、属性や世界観の核心で表現
