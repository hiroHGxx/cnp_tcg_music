# CNP UNIT PROJECT - 五彩リゾナンス企画書

*作成日: 2025-10-12*
*更新日: 2025-10-13*

## 📋 企画概要

**コンセプト**: CNP TCGの5人のキャラクター（ツミカ・ミライ・クルミ・リサ・ミナモ）が音楽ユニット「五彩リゾナンス」を結成するIF企画。各大陸の特色を活かした現代的エレクトロポップアルバムを制作。

**ターゲット**: CNP TCGファン、エレクトロポップファン、K-Pop/J-Popリスナー

**ユニット名**: 五彩リゾナンス（Gosai Resonance / 5-Color Resonance）

**制作形式**: Suno AIを使用した楽曲生成

**音楽的方向性**:
- **参考アーティスト**: Perfume（テクノポップ・エレクトロポップ）、K-Pop女性グループ（NewJeans, aespa, IVE等）
- **サウンド特徴**: 2020年代の現代的デジタルサウンド、重いサイドチェイン、ボコーダー・オートチューン多用
- **避けるべき要素**: 80-90年代アイドル風の温かみのあるアナログサウンド、ビンテージ感

## 🎤 制作方針

### **音楽的統一感**
- **現代的デジタルサウンド**: 2020年代の洗練されたエレクトロニックプロダクション
- **Perfume系テクノポップ**: 機械的で精密なビート、ボコーダー・オートチューン多用
- **K-Pop要素**: ヒップホップ・トラップビート、808ベース、激しいサイドチェイン
- 各キャラクターの個性を音楽で表現
- 既存CNP楽曲との差別化（戦闘→アイドル）

### **サウンド指針（重要）**

#### **✅ 取り入れるべき要素**
- **プロダクション**: ハイパーポップ影響のラウドマスタリング（-8 to -10 LUFS）、デジタルで鋭い音質
- **ボーカル処理**: オートチューン・ピッチ補正（スタイライズド）、ボコーダー、ボーカルチョップ、ロボティックテクスチャ
- **ベース**: 808サブベース（ディストーション付き）、サイドチェインで強くパンピング
- **シンセ**: スーパーソー（デチューン・ワイドステレオ）、デジタルプラック、グリッチエフェクト
- **ドラム**: トラップスタイル（レイヤードクラップ・スネア）、クリスプで鋭い
- **エフェクト**: ライザー、インパクト、ボーカルチョップ、スタッター、デジタルグリッチ

#### **❌ 避けるべき要素**
- ビンテージ・アナログの温かみ（80-90年代サウンド）
- 生楽器の質感（アコースティックギター等）
- オーガニックで自然なボーカル（加工なし）
- ローファイ・チルサウンド

### **歌詞の方向性**
- ポジティブで希望に満ちたメッセージ
- 各キャラクターの成長・絆・夢
- CNP世界観の用語を自然に織り込む
- ファンが共感できる普遍的テーマ

### **ボーカル設定**
- 全曲女性ボーカル（5人ユニット設定）
- **重要**: オートチューン・ピッチ補正を全曲に適用（Perfume風）
- ユニット曲: 5人の個性が活きるパート分け + ボコーダーハーモニー
- ソロ曲: 各キャラクターの性格に合わせた歌唱スタイル + エフェクト処理
- 日本語メイン、英語タグ併用

### **Sunoプロンプト最適化**
- テンプレート使用（SUNO_MUSIC_PROMPT_TEMPLATE.md）
- **目標文字数**: 500-600文字（余裕を持たせる）
- 870文字以内圧縮ルール厳守
- カード根拠を明記
- アーティスト名禁止、ジャンル特徴で表現
- **必須キーワード**: "2020s", "K-Pop", "electropop", "hyperpop", "digital", "sidechain", "vocoder", "autotune"
- **圧縮テクニック**: 略語使用（M1-M5）、冗長表現削除、接続詞・冠詞削除、改行最小化
